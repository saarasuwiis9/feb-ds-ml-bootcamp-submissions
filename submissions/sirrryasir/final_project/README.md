# AgriTech AI | AI-Based Crop Recommendation System

**Name:** Yasir Hassan Ahmed

## Short Description
AgriTech AI is an end-to-end Machine Learning ecosystem designed to recommend the most optimal crop for a given soil based on NPK, pH, and weather parameters. To ensure maximum precision (>99%), it employs advanced ML techniques such as `GridSearchCV` hyperparameter tuning and K-Fold cross-validation on an underlying Random Forest Classifier. The model is deployed via a fast Flask REST API and provides an interactive, modern user experience through a deployed Next.js dashboard.
